# Amount-Parser
## Translates a number into its text based representation.

### Requirements:

- [Leineghen](http://leiningen.org/#install)
- [Java](https://java.com/en/download/index.jsp) 

### Installation:

`$ lein deps`

### Tests:

`$ lein test`

### Interactive Mode:

`$ lein run`
